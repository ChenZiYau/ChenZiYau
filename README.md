## Featured Project

### OptiPlan — AI-Powered Productivity Dashboard

OptiPlan is my Final Year Project, focused on helping students and young professionals manage their productivity in one unified platform.

Instead of switching between multiple separate tools, OptiPlan aims to combine key productivity features into a single dashboard.

```txt
Core Features:
- Smart scheduling
- Personalized study tools
- Budget tracking
- Team collaboration
- Productivity insights
- AI-assisted planning
```

The main goal of this project is to reduce workflow fragmentation and help users manage their tasks, study plans, and personal productivity more efficiently.
